Java Animator

A program that animates a text file of keyframes using MVC design.

Description

This program was developed to read a file of keyframe instructions, interpret them, and format them to either output a text file of the different shapes and instructions each shape had in the animation, a .svg file that would open a browser and play the animation, or an editor made with JPanel where the user could play the animation, add and animate additional shapes, or delete existing shapes or animation instructions.

Details

Programming Language: Java
Role: Programmer
Source code available upon request.

Contributions

  • Collaborated to design methods and functionality of the MVC implementation
  • Developed objects to correspond to keyframes, shapes, motions, and positions
  • Created tool to adapt object shapes into JPanel shapes for displaying in the editor

The Process

This project was done in collaboration with another student over the course of a few weeks between three different assignments. The first assignment was about developing a solid design for reading keyframes and turning them into recorded objects. My partner and I had difficulties at first creating a good design for this, but once we had one that the professor and TAs gave the thumbs-up, the rest was much easier. We were then able to develop a way of exporting the recorded keyframes into text files and svg files, and finally we were able to process them into an animation editor we built using JPanel.
